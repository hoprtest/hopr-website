WHAT IS HOPR?

For cryptographers:

HOPR is a Chaumian mixnet with onion encryption. On the message layer HOPR is implementing a modified version of the Sphinx package format.

For developers:

Check out HOPR and contribute on GitHub we ♥ open source

For miners:

Earn money by running a HOPR relay node relaying messages of other participants (for now testnet money only ;-)

For investors:

Join the journey of HOPR towards a decentralized & privacy-preserving web 3.0. Get in touch with Sebastian to find out more.

How does it work?

We had the honor of presenting and demoing HOPR at EthCC2019 in Paris. If you missed it check out the video here.

TLDR pls?

HOPR is a privacy-preserving messaging protocol that incentivizes users to participate in the network. It provides privacy by relaying messages via several relay nodes to the recipient. Relay nodes are getting paid via payment channels for their services.

Where does it live?

HOPR loves GitHub, check out our repository. The bootstrap node is running at hoprnet.io (just like this website). HOPR is supported by the HOPR Association registered in Zurich, Switzerland.
